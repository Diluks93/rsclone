# rsclone
This repository is the clone game  "How to annoy neighbor".

## Play Online

- Browser: https://diluks93.github.io/rsclone/index.html
- Keyboard
  - `Cursor keys`: Movement
    - `A/D key`: Move Left/Right,
    - `E key`: Use of things
  - `SPACE`: Use the door
- Mouse Controls
  - Use the SPA part of the app and skip the tutorial
